knowledge linker

Agent 系统

严格符合 Agent 四要素的知识管理系统：感知 → 规划 → 工具调用 → 反馈闭环。

快速开始

pip install -r requirements.txt
cp .env.example .env
python main.py
python main.py --feedback starred

Agent 能力

自动发现、链接、整理零散知识碎片，构建结构化知识图谱。支持用户反馈学习（starred / clicked / dismissed）。

技术栈

Python · Anthropic API · 知识图谱 · SQLite
