knowledge linker

Agent 系统

基于 LangGraph 构建的知识管理 Agent MVP，手动输入驱动，自动完成语义关联发现与主动推送，实现感知 → 规划 → 工具调用 → 反馈闭环四要素。

部署方式

python deploy_server.py   # 端口 8710，PORT 环境变量可改

对接方式

iframe 嵌入 — <iframe src="http://你的服务器:8710">

API 调用 — 主站搜索框调 /api/recall 搜知识库

Nginx 代理 — 绑定子域名指向服务

技术栈

Python · LangGraph · Anthropic API · 知识图谱 · 语义关联
